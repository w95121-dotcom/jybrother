# 周易时间起卦工具 MVP

一个基于 Next.js、TypeScript 和 Tailwind CSS 的梅花易数时间起卦工具。

## 本地运行

```bash
npm install
npm run dev
```

## 测试

```bash
npm test
```

## 部署到 Vercel

1. 将 `zhouyi-time-gua` 目录提交到 GitHub、GitLab 或 Bitbucket 仓库。
2. 登录 [Vercel](https://vercel.com)，选择 **Add New Project**。
3. 导入仓库，项目根目录选择 `zhouyi-time-gua`。
4. Framework Preset 选择 **Next.js**。
5. Build Command 使用 `npm run build`，Install Command 使用 `npm install`。
6. 点击 **Deploy**。
